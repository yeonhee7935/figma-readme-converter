사용 방법

이 플러그인은 Markdown 형식의 텍스트를 Figma 프레임으로 변환합니다.

지원하는 Markdown 구문

  • 헤딩: # 제목, ## 부제목, ### 소제목
  • 목록: - 항목, * 항목, + 항목 (중첩 목록 지원)
  • 페이지 구분: --- (구분선)

팁

  • 헤딩은 각 섹션의 시작을 나타냅니다.
  • 구분선(---)은 새로운 프레임을 생성합니다 (Instagram 캐러셀용).
  • 설정 탭에서 프레임 크기, 글꼴, 색상 등을 사용자 지정할 수 있습니다.
